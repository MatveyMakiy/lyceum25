Описание программы:
Программа анализирует текущее состояние игрового поля в игре "крестики-нолики" и делает наилучший ход за игрока крестики. После хода выводит обновлённое поле и комментарий к действию.

Формат ввода:
Пользователь вводит поле в виде трёх строк по три символа в каждой, это ввод как бы подразумивает клетки, границы клеток обозначены квадратными границами '[' и ']', в данных границах пользователь вводит 3 варианта символов ' ' - пустая клетка, 'X' - клетка с крестиком и '0' - клетка с ноликом.

Пример ввода:
[X][ ][ ]
[X][0][ ]
[ ][ ][0]

Гарантируется, что поле для игры будет иметь корректный вид. В противном же случае программа уведомит пользователя о некорректном вводе и перестанет работать.

Программа определяет ход по следующей иерархии приоритетов:
1. Если есть ход, побеждающий за 'X' — сделать его. 
2. Если у '0' есть выигрышный ход — заблокировать его. 
3. Если центральная клетка (1,1) свободна — занять её.
4. Занять один из свободных углов (0,0), (0,2), (2,0), (2,2). 
5. Занять любую свободную клетку.

Программа должна найти наилучший ход для данной конфигурации игрового поля. После хода выводит игровое поле и подводит итог своим дествиям.
