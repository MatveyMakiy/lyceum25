Техническое задание: Игра "Жизнь" с графическим интерфейсом

1. Цель проекта
Создать графическое приложение для симуляции клеточного автомата "Игра Жизнь" Джона Конвея с возможностью интерактивного задания начальных условий, настройки параметров и визуализации эволюции клеточных структур.

2. Функциональные требования
    2.1. Основная функциональность
        - Симуляция классических правил игры "Жизнь" (23/3)
        - Визуализация игрового поля в реальном времени
        - Пошаговое выполнение и автоматический режим
        - Возможность приостановки и возобновления симуляции

    2.2. Способы задания начальной конфигурации
        - Ручное рисование клеток мышью
        - Случайное заполнение поля с заданной плотностью
        - Загрузка конфигурации из текстового файла
        - Сохранение текущего состояния в файл

    2.3. Настройки игрового поля
        - Выбор размера поля из предустановленных вариантов
        - Автоматическое масштабирование клеток под выбранный размер
        - Циклические границы (тороидальное пространство)

3. Валидация и обработка ошибок
    3.1. Проверка входных данных
        - Корректность формата загружаемых файлов
        - Проверка границ массива при рисовании
        - Обработка отсутствующих файлов

    3.2. Обработка системных ошибок
        - Нехватка памяти при создании больших полей
        - Ошибки ввода-вывода при работе с файлами
        - Графические ошибки отрисовки

4. Интерфейс
    4.1. Графический интерфейс
        - Основное игровое поле с клеточной сеткой
        - Панель управления с кнопками
        - Информационная строка состояния
        - Настраиваемые визуальные параметры

    4.2. Элементы управления
        - Кнопки: "Очистить", "Шаг", "Старт", "Стоп"
        - Переключатели режимов ввода начальных условий
        - Ползунок регулировки скорости симуляции
        - Выпадающий список выбора размера поля

    4.3. Визуальное представление
        - Живые клетки: черные квадраты
        - Мертвые клетки: белые квадраты с серой сеткой
        - Размер клеток адаптируется под размер поля

    4.4. Интерактивные возможности
        - Рисование/стирание клеток кликом мыши
        - Заливка при зажатой кнопке мыши
        - Автоматическая остановка при стабилизации

5. Требования к данным
    5.1. Формат файлов
        - Текстовые файлы с расширением .txt
        - Символы: '1' или 'X' для живых клеток, '0' или пробел для мертвых
        - Каждая строка - ряд поля, символы без разделителей

    5.2. Размеры поля
        - Предустановленные размеры: 30x25, 40x30, 50x40, 60x50, 70x60
        - Автоматический подбор размера клеток
        - Сохранение пропорций отображения

    5.3. Параметры случайного заполнения
        - Три уровня плотности: 30%, 50%, 70%
        - Равномерное случайное распределение
        - Независимые вероятности для каждой клетки

6. Ограничения
    - Максимальный размер поля: 70x60 клеток
    - Минимальная задержка между шагами: 10 мс
    - Максимальная задержка между шагами: 500 мс
