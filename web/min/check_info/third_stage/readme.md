На момент завершения данного этапа реализована базовая серверная часть проекта. Была создана структура backend, настроен запуск приложения и установлены необходимые библиотеки для работы с сервером и базой данных. Для взаимодействия с базой данных используется Prisma, проведена настройка подключения и выполнены миграции. Реализована система аутентификации пользователей: доступны регистрация, вход в аккаунт и получение данных текущего пользователя. Для этого используется JWT-токен, а также добавлен middleware для проверки авторизации при обращении к защищённым маршрутам. Добавлены основные сущности проекта — группы и посты. Реализованы основные endpoints для работы с ними: получение списка, создание, а также (для постов) редактирование и удаление. Все запросы были протестированы с помощью Bruno и работают корректно. Форматировщик и линтер настроены и, в даленейшем, пройдут доработку, если такое будет нужно

Таким образом, данная часть была интегрирована с фронтендом, созданным на втором этапе, в дальнейшем будет расширен для использования уже во всем
