# Lyceum Social Network

Финальный проект по веб-разработке за 2 полугодие 10 класса.

Проект представляет собой небольшую социальную сеть для лицеистов. В приложении можно регистрироваться, входить в аккаунт, создавать публикации, вступать в группы, оставлять комментарии, ставить лайки, создавать мероприятия, записываться на них, переписываться с пользователями и пользоваться админкой.

## Основные возможности

### Пользовательская часть

- регистрация и вход;
- выход из аккаунта;
- просмотр ленты;
- отображение публичных мероприятий для гостей;
- отображение постов только из групп, на которые подписан пользователь;
- создание личных публикаций;
- создание публикаций в группах;
- лайки публикаций;
- комментарии к публикациям;
- поиск групп;
- поиск пользователей;
- просмотр чужого профиля;
- редактирование своего профиля;
- личные сообщения;
- групповые чаты;
- светлая и тёмная тема.

### Группы

- создание группы;
- редактирование группы;
- страница группы;
- вступление в группу;
- выход из группы;
- роли в группе:
  - администратор;
  - модератор;
  - участник;
- назначение и снятие модераторов;
- создание постов группы;
- чат группы.

### Мероприятия

- создание мероприятий в группе;
- редактирование мероприятий;
- удаление мероприятий;
- запись на мероприятие;
- отмена записи;
- публичные мероприятия от администрации;
- отображение ближайших мероприятий в ленте.

### Админка

Глобальный администратор может:

- просматривать статистику сайта;
- создавать публичные мероприятия;
- удалять посты;
- удалять комментарии;
- удалять группы;
- удалять мероприятия;
- удалять пользователей;
- назначать других пользователей глобальными администраторами;
- снимать роль глобального администратора.
