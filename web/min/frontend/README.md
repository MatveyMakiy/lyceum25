# Социальная сеть для лицея — фронтенд

## О проекте

Это фронтенд-часть учебного проекта — социальной сети для учеников и преподавателей лицея.  
Приложение проектируется как единая система с лентой публикаций, профилями пользователей, группами, сообщениями и событиями.

## Что реализовано на этапе 2

На текущем этапе реализовано:

- структура фронтенд-проекта;
- настройка запуска через Vite;
- страницы входа и регистрации;
- базовая валидация форм;
- mock API для авторизации и ленты;
- страница ленты публикаций с пагинацией;
- защита приватных страниц для неавторизованных пользователей;
- базовые заглушки для будущих страниц:
  - профиль,
  - группы,
  - сообщения;
- базовые unit-тесты через Vitest.

## Используемые технологии

- HTML
- CSS
- JavaScript
- Vite
- Vitest

Установить зависимости:
